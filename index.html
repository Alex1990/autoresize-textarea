autoresize-textarea

A jQuery (optional) plugin can automatically resize the textarea's height.

Demo

Try to input or delete some characters.

  • Input interactions: Type, Paste (Ctrl+X or Context Menu), Drag and Drop, Undo (Ctrl+Z or Context Menu).
  • Delete interactions: Backspace key, Delete key, Cut (Ctrl+X or Context Menu), Drag and Drop, Undo (Ctrl+Z or Context menu).
